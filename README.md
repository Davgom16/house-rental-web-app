# ArtHands Marketplace

ArtHands Marketplace is a full-stack web application that connects local artisans with customers looking for handmade and unique products. The platform allows users to browse products, add items to a shopping cart, and manage their purchases in real time.

---

## Features

- Product listing from a MySQL database
- Shopping cart system (add, update, remove items)
- Real-time cart updates using REST API
- Product categories (wood crafts, knitting, sweets, drinks, etc.)
- Contact form with validation
- Login / Sign-up form validation (frontend)
- Responsive navigation layout

---

## Tech Stack

**Frontend:**
- HTML5
- CSS3
- JavaScript (Vanilla JS)

**Backend:**
- Node.js
- Express.js

**Database:**
- MySQL
